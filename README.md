# Sentiment Analysis using Python

## 📌 Project Overview

This project performs **Sentiment Analysis** on text data using Python.
The program analyzes reviews and classifies them as **Positive, Negative, or Neutral** using Natural Language Processing (NLP).

Sentiment Analysis helps understand public opinion from sources like product reviews, social media posts, and feedback.

---

## 🎯 Objective

* Analyze text reviews
* Identify sentiment (Positive / Negative / Neutral)
* Use Python NLP libraries for sentiment classification

---

## 🛠 Tools and Technologies

* Python
* TextBlob
* VS Code
* Natural Language Processing (NLP)

---

## 📂 Project Files

* `sentiment_analysis.py` → Python program for sentiment analysis
* `README.md` → Project documentation

---

## ⚙️ Installation

Install the required library:

pip install textblob

Then download required data:

python -m textblob.download_corpora

---

## ▶️ How to Run the Project

1. Open the project in **VS Code**
2. Open terminal
3. Run the program

python sentiment_analysis.py
